# Portfolio Pulse - 全球资产仪表盘

**Version:** 1.0.0
**Type:** Flagship Skill
**Domain:** Wealth Management
**Target Users:** 高净值人群、投资者

## Features

- **多资产追踪** - 股票、加密货币、大宗商品、外汇
- **实时估值** - 持仓市值、盈亏计算
- **风险分析** - 集中度、分散度评分
- **再平衡建议** - 智能调仓建议
- **表现分析** - 周/月/年收益对比

## Usage

```
"我的投资组合表现如何"
"资产配置分析"
"苹果股票现在多少钱"
"比特币价格"
```

## Supported Assets

| 类型 | 示例 |
|------|------|
| 美股 | AAPL, MSFT, GOOGL, TSLA |
| 港股 | 0700.HK (腾讯), 9988.HK (阿里) |
| 加密货币 | BTC, ETH, DOGE |
| 大宗商品 | 黄金, 白银, 原油 |
