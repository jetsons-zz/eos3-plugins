# Executive Travel Intelligence - 高管出行智囊

旗舰级商务出行助手，6维度融合分析，一句话获取全方位出行决策支持。

## Features

- **天气预报** - 7天预报、穿衣建议、行李提醒
- **空气质量** - AQI、健康建议、口罩提醒
- **汇率换算** - 实时汇率、预算估算
- **时区转换** - 时差计算、最佳会议时间
- **节假日** - 当地节假日、工作日检查
- **智能评分** - 0-100出行评分、风险提示

## Usage

```
"我下周去东京出差"
"去伦敦出差3天，帮我分析"
"纽约出行报告"
```

## Supported Cities

东京、北京、上海、香港、新加坡、首尔、伦敦、巴黎、纽约、洛杉矶等16个主要商务城市
