# Global Market Pulse - 全球股市快报

为高净值人群和企业高管提供全球市场实时行情与专业分析。

## Features

- **全球股指** - 标普500、道琼斯、纳斯达克、恒生、上证等14个主要指数
- **市场情绪** - 多空判断、VIX恐慌指数分析
- **汇率行情** - 美元、欧元、人民币等主要货币
- **大宗商品** - 黄金、白银、原油价格
- **加密货币** - 比特币、以太坊行情
- **高管简报** - 一句话市场摘要

## Installation

